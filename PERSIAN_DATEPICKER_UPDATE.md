# تغییرات Date Picker به فارسی

## تغییرات انجام شده

تمام date pickerهای معمولی در برنامه به date picker فارسی تبدیل شدند.

### فایل‌های تغییر یافته:

#### 1. `src/components/SimplePersianDatePicker.jsx` (جدید)
- یک نسخه ساده‌شده از PersianDatePicker
- با فرمت ISO تاریخ کار می‌کند
- تبدیل خودکار بین تاریخ میلادی و شمسی
- قابل استفاده در همه جای برنامه

#### 2. `src/pages/Attendance.jsx`
تغییرات:
- Import کردن `SimplePersianDatePicker` به جای date picker معمولی
- جایگزینی date picker انتخاب تاریخ در هدر صفحه
- جایگزینی date pickerهای فیلتر بازه زمانی (از تاریخ - تا تاریخ)

### فایل‌هایی که از قبل از PersianDatePicker استفاده می‌کردند:

✅ `src/components/TransactionForm.jsx` - استفاده از PersianDatePicker
✅ `src/components/MemberForm.jsx` - استفاده از PersianDatePicker

## ویژگی‌های SimplePersianDatePicker:

1. **تقویم شمسی کامل**
   - نمایش ماه‌های فارسی
   - نمایش روزهای هفته فارسی
   - مشخص ��ردن جمعه‌ها با رنگ قرمز

2. **انتخاب سال و ماه**
   - منوی dropdown برای انتخاب سریع ماه
   - منوی dropdown برای انتخاب سریع سال (1340-1420)

3. **تبدیل خودکار تاریخ**
   - تبدیل تاریخ میلادی به شمسی برای نمایش
   - تبدیل تاریخ شمسی به میلادی برای ذخیره
   - سازگار با فرمت ISO

4. **موقعیت هوشمند**
   - باز شدن به سمت بالا یا پایین بسته به فضای موجود
   - جلوگیری از خروج از صفحه

5. **بستن خودکار**
   - بسته شدن با کلیک خارج از تقویم
   - بسته شدن بعد از انتخاب تاریخ

## نحوه استفاده:

```jsx
import SimplePersianDatePicker from "../components/SimplePersianDatePicker";

// در کامپوننت:
<SimplePersianDatePicker
  value={selectedDate}
  onChange={(date) => setSelectedDate(date)}
  className="w-full px-4 py-2 border rounded-lg"
/>
```

## نکات مهم:

1. **فرمت تاریخ**: تاریخ‌ها به صورت ISO (YYYY-MM-DD) ذخیره می‌شوند
2. **نمایش**: تاریخ‌ها به صورت شمسی (YYYY/MM/DD) نمایش داده می‌شوند
3. **سازگاری**: کاملاً با سیستم قبلی سازگار است
4. **تبدیل تقریبی**: تبدیل تاریخ به صورت تقریبی است (621 سال اختلاف)
